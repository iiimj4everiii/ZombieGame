This is for demo only.
Only level1.txt will be used for game map.

Use W,A,S,D to move around the game world.
Use mouse pointer and left click to point and shoot.
Use 1, 2, and 3 to switch between different guns.

1. Pistol - Highest damage per bullet, smallest spread, slower firing rate.
2. Shotgun - Smallest damage per bullet, highest spread, slowest firing rate.
3. Sub-Machine gun - smaller damage per bullet, smaller spread, fastest firing rate.

Game file: ZombieGame.exe in Release folder.
